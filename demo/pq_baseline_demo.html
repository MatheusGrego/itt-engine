P_eq as Geometric Counterfactual Baseline

Visual validation — ITT applied to a 2D toy black hole
This is a visual demonstration of a reformulation of P_eq in Informational Tension Theory (ITT). Instead of defining the baseline as uniform maximum entropy, define it as the same system with the perturbation source removed. Here:

P_obs = light rays under gravity (deflected)  ·  P_eq = same rays without gravity (straight lines)  ·  τ = JSD(P_obs ∥ P_eq)

Toggle the layer buttons to view each field. The τ peak should coincide with the photon sphere (orange dashed ring) — a geometrically invariant feature, not an artifact of how rays were injected.

Click on the canvas to launch a single ray going right. Click and drag to set a custom direction and speed (drag distance = speed). Each launched ray creates both an observed (with gravity) and expected (without gravity) twin.
